# SignVoice Features

## Feature 1: Real-Time Sign Language to Speech and Text
- Camera watches user signing
- AI recognizes signs instantly
- Shows text on screen + speaks out loud
- Laptop microphone picks up speech for meetings

## Feature 2: Multi-Language Support
Four combinations:
- FSL → Tagalog
- FSL → English
- ASL → English
- ASL → Tagalog

## Feature 3: Live Conversation Mode (Two-Way)
- Deaf person signs → AI shows text + speaks
- Hearing person speaks → AI shows text
- Both appear in conversation thread
- Blue bubbles = signs, Gray bubbles = speech

## Feature 4: Meeting Integration (Companion App)
- Works alongside any meeting platform
- Phone sits next to laptop
- Phone speaks, laptop mic picks it up
- Meeting participants hear the deaf person
- Phone mic picks up meeting audio
- No installation needed for others

## Feature 5: Context Awareness
Resolves ambiguous Tagalog words:
- BASA: read vs wet
- BUKAS: tomorrow vs open
- SUKA: vinegar vs vomit
- PUSO: heart (organ) vs heart (love) vs banana blossom
- PATAY: dead vs turn off

Uses 4 clues:
1. Surrounding signs
2. Facial expression
3. Conversation history
4. Common probability
